# Homebrew Tap for Bugster

This is a Homebrew tap for installing [Bugster CLI](https://github.com/Bugsterapp/bugster-cli).

## Installation

### Direct installation (recommended)

```bash
brew install Bugsterapp/homebrew-tap/bugster
```

### Manual tap installation

```bash
brew tap Bugsterapp/homebrew-tap
brew install bugster
```

## Usage

After installation, you can use the `bugster` command:

```bash
bugster --version
```

## Updating

To update Bugster to the latest version:

```bash
brew upgrade bugster
```
